# ⚛️ Modelos Atômicos Interativos

> Visualizador interativo da evolução histórica dos modelos atômicos — otimizado para celular, sem dependências externas.

**🇧🇷 Português** | [🇪🇸 Español](#-modelos-atómicos-interactivos)

---

## 📱 Demo

Baixe o arquivo `modelos_atomicos.html` e abra diretamente no navegador do celular — funciona 100% offline.

---

## 🔬 Modelos incluídos

| Modelo | Ano | Cientista | Destaque da simulação |
|---|---|---|---|
| **Dalton** | 1808 | John Dalton | Esfera maciça com gradiente 3D |
| **Thomson** | 1897 | J.J. Thomson | Elétrons com física de colisão elástica real |
| **Rutherford** | 1911 | E. Rutherford | Órbitas elípticas com velocidade Kepleriana |
| **Bohr** | 1913 | Niels Bohr | Distribuição real por camadas K, L, M, N |
| **Quântico** | 1926 | Schrödinger / Heisenberg | 7 orbitais com nuvem de probabilidade |

---

## 🌌 Orbitais do Modelo Quântico

| Orbital | Forma | Descrição |
|---|---|---|
| `1s` | Esférico | Distribuição gaussiana isotrópica |
| `2p` | Dois lobos | Densidade concentrada nos polos |
| `3d z²` | Bilobular + toro | Dois lobos axiais com anel equatorial |
| `3d xy` | Quatro lobos (45°) | Lobos entre os eixos |
| `3d x²-y²` | Quatro lobos (eixos) | Lobos alinhados aos eixos |
| `4f z³` | Seis lobos axiais | Estrutura complexa com anéis laterais |
| `4f xyz` | Oito lobos cúbicos | Distribuição octante |

---

## ⚙️ Funcionalidades

- **Abas deslizáveis** para navegar entre modelos
- **Slider de elétrons** (1–18) com distribuição automática por camadas
- **Slider de velocidade** das animações
- **Pausar / Reset** da animação
- **Cards didáticos** com cientista, ano, conceito e descrição histórica
- **Glow e gradientes** nos elétrons e núcleos
- **Rastro de partículas** no modelo de Rutherford
- **Flash de colisão** no modelo de Thomson

---

## 🛠️ Tecnologias

- HTML5 puro — sem frameworks, sem bibliotecas externas
- Canvas 2D API para todas as animações
- CSS com variáveis e layout responsivo
- JavaScript vanilla com física de partículas

---

## 🎓 Contexto educacional

Desenvolvido como material didático para o ensino de Química no nível médio, com foco em:

- Evolução histórica dos modelos atômicos
- Visualização de conceitos abstratos (orbitais, probabilidade)
- Uso pedagógico de IA com **human in the loop** — a IA acelera a criação, o professor orienta, o aluno protagoniza

> *"Há 26 anos, criar uma página assim levava meses e uma equipe inteira. Hoje, em 30 minutos, temos um resultado que vai direto para as mãos dos alunos."*
